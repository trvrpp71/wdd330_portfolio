FETCH API

fetch() replaces XML term

fetch() returns a promise

Basic syntax

let promise = fetch(url, [options])

the promise resolves with the built-in response class

Example

let response = await fetch(url);
if (response.ok) { // if HTTP-status is 200-299
// get the response body (the method explained below)
let json = await response.json();
} else {
alert("HTTP-Error: " + response.status);
}

a typical fetch consists of either:

two await calls:

let response = await fetch(url, options); // resolves with response headers
let result = await response.json(); // read body as json

OR

wihtout the await call:

fetch(url, options)
.then(response => response.json())
.then(result => /* process result */)

Response Properties

  • response.status - HTTP code of the response
  • resonse.ok - true ; status is 200-299
  • response.headers - map-like object with HTTP headers

    • methods to the response body

  • response.text() - returns response as text
  • response.json() - parse teh response as JSON object
  • response.formData() - return the response as formdata object
  • response.blob() - return the response as Blob binary data type
  • response.arrayBuffer() -return response as ArrayBuffer
