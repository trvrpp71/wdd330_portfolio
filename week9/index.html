Novice to Ninja: Chapter 9 Summary

    - the window object is global object in a browser

    - global variables belong to the window object

    - alert, confirm() and prompt() are all methods of the window object and open dialogs that stop the execution of a program

    - window.navigator object give info about the users browser and OS...can be unreliable

    - window.location object provides info about hte URL of the current page

    - window.history object keeps info about the pages visited in the current session

    - window object can open, close, resize and move windows, but you shouldn't

    - window.screen object provides information about the users screen

    - document.write() is an old method of writing text to the document and should be avoided

    - cookies can be used to store small pieces of info between requests using the document.cookie property

    - window.setTimeout() method can be used to invoke a function after a set amount of time has passed. To cancel it use the clearTimeout() method.

    - window.setInterval() method can be used to repeatedly invoke a function. Stop the invocation using the clearInterval() method

    - window.requestAnimationFrame() method can be used to produce smooth and optimized animations by utilizing the browsers built-in graphics capabilities. Cancel it by using the cancelAnimationFrame() method.

Novice to Ninja Chapter 14 Summary

    HTML5 is the latest and greatest of the markup language

    data- attributes help to embed custom data into a web page that can be used with JS to enhance the experience

    Web Storage API allows key-value pairs to be stored on the users device similar to cookies but without the same restrictions

    Geolocation API allows one to access the geographical coordinates of the users device, with permission

    Web Worker API can be used to perform computationally heavy tasks in the background to help avoid unresponsivness

    Web sockets are a new protocol for communicating over the internet, allowing two-way communications

    Notifcation API displays notifications on the user system

    audio & video elements can be used to embed audio and video clips into a web page. The also have a Media API that can help control the playback using JS.

    canvas element can be used to dynamically draw geometric shapes, text and images on the a web page in real-time using JS

    A shim or polyfill is a piece of code that allows support of missing features to older browsers
